============ Social_Home ============

Social_Home is a web service that will provide easy control over your house applications in the web, basically some home automation together with your favorite social media. 

we will be using:
- facebook React
- facebook Parse
- facebook LogIn
- .club domains

hardware:
-Raspberri Pi
-Arduino
-Servos
-Web cam
-LEDs
